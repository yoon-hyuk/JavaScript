        자바스크립트는 싱글쓰레드 언어이다. 어떻게 동시성을 구현할까?
        자바스크립트 인터프리터는 싱글쓰레드로 자바스크립트의 코드를 읽는다.
        중간에 비동기 동작해야하는 코드를 만나면, 해당 코드의 수행은 브라우저의 엔진에게 실행을 위임
        브라우저가 비동기적으로 수행되어야하는 작업을 병렬로 처리하고 난 뒤,
        callback query에 비동기 작업이후 실행되어야하는 callback함수 객체를 넣어준다.
        만약 call stack에 수행해야할 작업이 없다면, 자바스크립트가 callback 함수객체를 callback query에서 가져와 수행

2. Promise

            Promise는 자바스크립트의 내장객체로 callback Hell을 해결하기 위해 등장.
            Promise는 비동기작업을 수행하고, 직업의 성공/실패 여부에 따라 then, catch 메서드를 메서드체인방식으로 사용할 수 있는 객체

            Promise의 상태
                1. pending      :   Promise의 연산이 완료되지 않는 상태
                2. fullfilled   :   Promise의 연산이 성공적으로 완료된 상태, resolve() 함수가 호출된 상태를 의미
                3. rejected     :   Promise 연산에 실패한 상태 / reject() 메서드가 호출된 상태를 의미

            Promise의 메서드
                1. resolve 메서드 : Promise의 상태를 fullfilled 상태로 만들고, 연산의 결과값을 then 메서드로 전달
                2. reject  메서드 : Promise의 상태를 rejected 상태로 만들고, 에러 객체를 catch메서드로 전달.

            Promise의 후속처리 메서드
                1. then : resolve메서드를 통해 전달받은 결과값을 사용해, 비동기 이후의 후속처리 연산을 수행하고, Promise 객체를 반환
                2. catch : reject메서드를 통해 전달 받은 에러 객체를 사용해 비동기 이후의 예외처리를 수행하고, Promise 객체를 반환
                3. finally : Promise의 fullfilled, reject 여부에 상관없이 반드시 실행되어야하는 코드를 작성하는 메서드
            
            Promise 구문
                1. Promise 객체를 생성할 때, 생성자의 매개변수에 callback함수를 등록, 
                    callback함수에서 resolve, reject함수객체를 받을 수 있도록 매개변수를 2개 선언
                2. then 메서드의 경우 콜백함수의 매개변수에 Promise의 reslove에 넘겨준 값이 넘어오게 된다.
                    then 메서드가 return 값이 있을 경우, 새로운 Promise를 반환한다.
                    then 메서드의 return이 Promise객체를 명시하는 경우에는, 우리가 명시한 Promise가 반환된다.

        asyn/await
        Promise 객체를 동기식 코드처럼 사용하기 위해 등장.

        사용방법
        비동기 코드를 동기식 코드처럼 다루고 싶은 함수의 선언부 앞에, asny 예약어를 붙여준다
        async함수안세서, 비동기 동작해야하는 코드를 Promise 객체를 반환하는 함수로 만들어주고, 해당 함수 앞에 await 예약어를 사용해준다.
        await예약어는 async 함수 내에서 반환되는 Promise가 pending상태를 벗어날 때 까지, async함수의 실행을 일시정지시킨다
        promise의 상태가 fullfield되면 promise가 pending상태를 벗어날 때 까지, async 함수의 실행을 일시정지시킨다
        rejected가 되면 reject된 값을 throw해준다.

        async함수는 반환값이 있을 경우 Promise 객체형태로 반환한다.
